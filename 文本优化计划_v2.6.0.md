## 6. 文本规范

### 6.1 命名规范
- **ID使用英文**: `event_id`, `npc_id`, `item_id`
- **显示使用中文**: 标题、描述、对话
- **注释使用中文**: 代码注释、文档说明

### 6.2 标点符号
- 使用中文标点：，。！？；：""''
- 避免英文标点：,.!?;:""''
- 特殊情况：代码中的标点保持英文

### 6.3 数字和单位
- 数字使用阿拉伯数字：1, 2, 3
- 单位使用中文：年、月、日、个、块
- 百分比：50%（数字+%）

### 6.4 专有名词
- 境界名称：炼气期、筑基期、金丹期
- 修行方向：剑修、体修、丹修、阵修
- 资源名称：灵石、修为、寿命
- 属性名称：声望、因果、关系

## 8. 检查工具

### 8.1 Python检查脚本
```python
# scripts/check_chinese_text.py
import json
import re

def check_english_text(text):
    """检查文本中是否包含英文单词"""
    # 排除常见的ID和代码
    if re.match(r'^[a-z_]+$', text):
        return False
    
    # 检查是否包含英文单词
    english_words = re.findall(r'\b[A-Za-z]{2,}\b', text)
    return len(english_words) > 0

def check_file(filepath):
    """检查JSON文件中的文本"""
    with open(filepath, 'r', encoding='utf-8') as f:
        data = json.load(f)
    
    issues = []
    
    # 递归检查所有文本字段
    def check_dict(obj, path=''):
        if isinstance(obj, dict):
            for key, value in obj.items():
                new_path = f"{path}.{key}" if path else key
                if isinstance(value, str) and key in ['title', 'description', 'text', 'name']:
                    if check_english_text(value):
                        issues.append({
                            'path': new_path,
                            'text': value
                        })
                else:
                    check_dict(value, new_path)
        elif isinstance(obj, list):
            for i, item in enumerate(obj):
                check_dict(item, f"{path}[{i}]")
    
    check_dict(data)
    return issues

# 使用示例
if __name__ == '__main__':
    files = [
        'data/events.json',
        'data/npcs.json',
        'data/quests.json'
    ]
    
    for filepath in files:
        print(f"\n检查文件: {filepath}")
        issues = check_file(filepath)
        if issues:
            print(f"发现 {len(issues)} 个问题:")
            for issue in issues[:10]:  # 只显示前10个
                print(f"  - {issue['path']}: {issue['text'][:50]}...")
        else:
            print("  ✓ 没有发现问题")
```
