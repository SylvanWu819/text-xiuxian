# 上传到 GitHub 的命令（按顺序执行）

## 第一步：提交到本地仓库
git commit -m "Initial commit: 修仙模拟器 v1.0.0"

## 第二步：连接到 GitHub 远程仓库
git remote add origin https://github.com/SylvanWu819/文字修仙.git

## 第三步：推送到 GitHub
git push -u origin main

---

注意：
1. 执行前请先在 GitHub 创建名为"文字修仙"的仓库
2. 不要勾选"Add a README file"
3. 首次推送可能需要输入 GitHub 用户名和密码（或 Personal Access Token）
