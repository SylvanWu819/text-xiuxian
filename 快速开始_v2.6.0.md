## ⚡ 常见问题

### Q1: 编译失败怎么办？
**A**: 
1. 检查Node.js版本（需要 >= 14）
2. 删除 `node_modules` 和 `package-lock.json`
3. 重新运行 `npm install`
4. 再次运行 `npm run compile`

### Q2: 游戏无法启动？
**A**:
1. 检查VSCode版本（需要 >= 1.80.0）
2. 重新加载窗口（Ctrl+Shift+P → Reload Window）
3. 查看开发者工具（F12）的错误信息
4. 检查是否有其他扩展冲突

### Q3: 存档丢失了？
**A**:
1. 检查是否运行了清理脚本
2. 查看 `data/backups/` 目录
3. 如果有备份，可以手动恢复
4. 如果没有备份，只能重新开始

### Q4: 发现Bug怎么办？
**A**:
1. 记录详细的重现步骤
2. 查看控制台错误信息
3. 在GitHub提交Issue
4. 或联系开发团队
