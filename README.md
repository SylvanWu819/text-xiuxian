# 修仙模拟器 (Cultivation Simulator)

一个运行在 VSCode 中的修仙文字游戏，体验选项驱动的修仙之旅。

## ✨ 特性

- 🎮 **丰富的游戏内容**：42个事件、56个NPC、70个任务
- 🌟 **多样化修炼路径**：剑道、炼体、炼丹、阵法等
- ⚖️ **因果系统**：善恶有报，影响游戏走向
- 🎲 **随机事件**：每次游戏都有不同体验
- 💾 **存档系统**：随时保存和加载游戏进度
- 🎨 **精美界面**：沉浸式的修仙体验

## 📦 安装

1. 在 VSCode 扩展市场搜索 "修仙模拟器"
2. 点击安装
3. 重启 VSCode

## 🎯 使用方法

1. 点击左侧活动栏的修仙图标
2. 选择"新游戏"开始修仙之旅
3. 或选择"加载游戏"继续之前的进度

## 🎮 游戏系统

### 修炼系统
- **境界**：练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期
- **修炼路径**：剑道、炼体、炼丹、阵法、音律等

### 因果系统
- 善行积累功德，恶行增加业力
- 影响NPC关系和事件触发

### 声望系统
- 正道声望、魔道声望
- 影响宗门态度和任务获取

### 资源管理
- 灵石：购买物品、学习功法
- 寿命：时间流逝会消耗寿命
