**位置**: `src/game/GameEngine.ts`  
**问题**: 选项需要道具时只检查不消耗  
**影响**: 游戏平衡性严重破坏  
**状态**: ✅ 已修复

**修复方案**:
```typescript
// 在executeTurn()中添加
if (option.requirements?.requiredItems) {
  for (const itemId of option.requirements.requiredItems) {
    this.consumeRequiredItem(itemId);
  }
}

// 新增方法
private consumeRequiredItem(itemId: string): void {
  // 按优先级从 items -> pills -> artifacts 消耗
}
```

#### Bug #2: 初始道具定义缺失 🔴
**位置**: `src/game/ResourceManager.ts`  
**问题**: healing_pill 和 spirit_herb 未定义  
**影响**: 显示为未知道具  
**状态**: ✅ 已修复

**修复方案**:
```typescript
// 在initializeResourceDefinitions()中添加
this.registerResource({
  id: 'healing_pill',
  name: '疗伤丹',
  type: ResourceType.Pills,
  rarity: ResourceRarity.Common,
  effect: { lifespanBonus: 1, description: '恢复伤势，延长1年寿命' }
});

this.registerResource({
  id: 'spirit_herb',
  name: '灵草',
  type: ResourceType.Pills,
  rarity: ResourceRarity.Common,
  effect: { description: '炼丹材料，可用于炼制丹药' }
});
```
