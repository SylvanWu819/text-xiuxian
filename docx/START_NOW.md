# 🚀 立即启动游戏

## ✅ 代码已编译完成！

现在按照以下步骤启动游戏：

## 🎮 启动步骤（只需 2 步）

### 第 1 步：按 F5 键

在当前 VSCode 窗口中按 **F5** 键

或者：
- 点击菜单：**运行 → 启动调试**
- 或使用命令面板（Ctrl+Shift+P）输入 "**Debug: Start Debugging**"

### 第 2 步：打开游戏

会打开一个新的 VSCode 窗口（扩展开发主机），在新窗口中：

1. **点击左侧活动栏的"修仙模拟器"图标**（太极图案）
2. 或使用命令面板（Ctrl+Shift+P）输入 "**修仙模拟器: 新游戏**"

## 🎯 就这么简单！

游戏界面会在侧边栏打开，你可以：
- 输入角色名称
- 选择修行方向（剑修/体修/丹修/阵修）
- 开始你的修仙之旅！

## ⚡ 快捷键

- **F5**：启动插件
- **Ctrl+Shift+P**：打开命令面板
- **Ctrl+Shift+X**：打开扩展面板

## 🐛 遇到问题？

### 问题：按 F5 没反应
- 确保在项目根目录（D:\WordGame）打开 VSCode
- 检查左下角是否显示"正在启动调试"

### 问题：找不到游戏图标
- 在新打开的窗口中查找左侧活动栏
- 图标是一个太极图案

### 问题：界面空白
- 按 F12 打开开发者工具查看错误
- 或查看 HOW_TO_START.md 了解详细排查步骤
