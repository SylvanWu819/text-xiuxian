# 功能添加：玩法说明

## 新增功能

在欢迎界面添加了"玩法说明"按钮，点击后显示详细的游戏玩法指南。

## 修改的文件

1. **src/extension.ts**
   - 在欢迎界面添加"玩法说明"按钮

2. **media/main.js**
   - 添加 `showHowToPlayPanel()` 函数，显示玩法说明面板
   - 在 `attachWelcomeListeners()` 中添加按钮事件监听器

3. **media/style.css**
   - 添加 `.how-to-play-panel` 样式
   - 添加 `.guide-section` 样式
   - 美化列表和标题样式

## 玩法说明内容

玩法说明面板包含以下章节：

### 🎮 游戏目标
介绍游戏的核心目标和玩法理念

### ⚔️ 修行方向
详细说明四种修行方向的特点：
- 剑修 - 攻击力强
- 体修 - 防御力高
- 丹修 - 资源丰富
- 阵修 - 控制力强

### 📊 核心属性
解释游戏中的关键属性：
- 修为 - 修炼境界
- 灵石 - 游戏货币
- 寿命 - 生存时间
- 声望 - 正道/魔道倾向
- 因果 - 善缘和因果债

### 🎯 游戏玩法
详细的游戏流程说明：
1. 选择行动
2. 时间推进
3. 随机事件
4. 境界突破
5. 渡劫系统
6. 人际关系
7. 势力系统

### 💡 游戏提示
给玩家的实用建议：
- 时间管理
- 寿命管理
- 因果管理
- 人际关系
- 修行方向优势
- 存档建议

### 🏆 结局系统
说明影响结局的因素：
- 修为境界
- 声望倾向
- 因果平衡
- 人际关系
- 特殊事件

### 🛠️ 工具栏功能
介绍工具栏各按钮的功能：
- 字体设置
- 存档功能
- 重开功能
- 历史记录

## 使用方式

1. 启动游戏后，在欢迎界面点击"玩法说明"按钮
2. 阅读详细的游戏指南
3. 点击"开始修仙"按钮关闭面板
4. 也可以点击面板外的遮罩层关闭

## 样式特点

- 使用 VSCode 深色主题配色
- 清晰的章节分隔
- 彩色标题和图标
- 列表项使用不同颜色的标记
- 关键词高亮显示
- 流畅的动画效果

## 技术实现

- 使用模态框（Modal）显示
- 动态创建 DOM 元素
- 事件委托处理关闭操作
- CSS 动画提升用户体验
- 响应式布局适配不同屏幕
