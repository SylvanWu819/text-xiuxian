# 修仙模拟器 v2.5.0 更新日志

## 版本信息
- **版本号**: v2.5.0
- **发布日期**: 2026-05-01
- **更新类型**: 功能更新

## 新增功能

### 背包与道具系统
- 新增背包系统，支持道具获取、使用和管理
- 新增道具需求检查，没有对应道具无法选择相关选项
- 新增道具效果系统，道具可影响修为、寿命、灵石等属性
- 新增道具获取渠道，探索事件可获得道具奖励
- 新增道具管理方法：`addItem()`, `removeItem()`, `hasItem()`, `getItemCount()`, `getAllItems()`, `getInventorySummary()`

## 技术改进
- 更新 `PlayerState` 接口，新增 `items: Map<string, number>` 字段
- 更新 `ResourceManager` 类，新增道具管理功能
- 更新 `GameEngine` 类，支持道具效果应用
- 更新 `OptionSystem` 类，支持道具需求检查
- 修复所有测试文件，确保编译通过

## 注意事项
- 本次更新为大版本更新，版本号从 v2.4.3 升至 v2.5.0
- 建议清理旧存档和缓存以确保功能正常
- 道具系统已完成基础实现，后续可根据需要添加UI显示
