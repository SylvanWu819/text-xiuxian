# 修仙模拟器 v2.2.0 更新日志

## 版本信息
- **版本号**: 2.2.0
- **更新日期**: 2026-04-30
- **更新类型**: 游戏内容简化

## 主要变更

### 游戏内容调整
- **简化日常修炼选项**: 现在只保留两个核心选项
  - 闭关修炼：消耗1个月时间，提升修为
  - 外出探索：消耗1个月时间，可能遇到机缘或危险

### 移除的功能
以下选项已从游戏中移除：
- 条件选项（突破境界、修炼方向专属选项等）
- 势力相关选项（宗门任务、加入宗门等）
- 资源相关选项（购买丹药、使用丹药、打工赚灵石等）

### 成就系统调整
由于部分游戏功能被移除，以下成就类别已暂时禁用：
- **门派声望类**：入门弟子、散修之路、叛出师门、宗门正义、寻回法器（共5个）
- **专业技能类**：炼丹大师、炼丹冠军、神兵铸造、雷霆神兵、傀儡操控、傀儡创造、酿酒大师（共7个）
- **战斗精通类**：剑道冠军、竞技场冠军（共2个）
- **名声显赫类**：傀儡工厂（1个）

**总计移除成就数**：15个
**剩余可用成就数**：约150+个

这些成就在未来版本中可能会重新启用或重新设计。

## 问题修复

### 修复隐居结局过早触发
- **问题**：玩家在金丹期就容易触发隐居结局，导致游戏过早结束
- **原因**：隐居结局的触发条件过于宽松
- **修复**：
  - 修为要求：金丹期 → 化神期
  - 声望要求：< 30 → < 20
  - 寿命要求：> 50% → > 60%
  - 新增时间要求：需要修炼至少300年
- **影响**：玩家可以继续修炼到更高境界，不会意外结束游戏

## 技术变更

### 代码修改
1. **OptionSystem.ts**
   - 简化 `generateOptions()` 方法，只生成基础选项
   - 保留 `generateBasicOptions()` 方法，仅包含"闭关修炼"和"外出探索"

2. **EndingSystem.ts**
   - 修复隐居结局触发条件过于宽松的问题
   - 提高触发要求：金丹期→化神期，声望<30→<20，寿命>50%→>60%
   - 新增时间要求：需要修炼至少300年

3. **版本更新**
   - package.json: 2.1.6 → 2.2.0
   - SaveSystem.ts: SAVE_VERSION 1.0.0 → 2.2.0
   - extension.ts: 欢迎界面版本显示更新为 v2.2.0

### 存档兼容性
- **重要**: 此版本不兼容旧存档
- 旧版本存档将无法加载
- 建议玩家开始新游戏

## 影响范围
- 游戏玩法更加聚焦于核心修炼和探索机制
- 简化了选项系统，降低了游戏复杂度
- 提升了游戏节奏，减少了选择负担
