# 战斗系统更新说明 v2.3.0

## 更新概述

新增**战斗力评估系统**，让战斗事件不再是纯随机，而是根据玩家实际状态动态计算胜负概率。

---

## 核心改进

### 1. 战斗力计算系统

玩家战力由以下因素决定：

#### 基础战力
- **修为等级**：炼气期100，筑基期300，金丹期800...
- **修为进度**：当前经验/最大经验，提供0-20%加成

#### 状态影响
- **寿命状态**：
  - 寿命<20%：战力-30%（重伤垂危）
  - 寿命<50%：战力-15%（状态不佳）
  - 寿命>90%：战力+10%（生机勃勃）

- **因果业力**：
  - 善缘多：战力+20%（天道眷顾）
  - 业力重：战力-20%（因果反噬）

#### 装备加成
- 普通法器：+15%
- 上品法宝：+25%
- 破损仙器：+30%

#### 体质加成
- 雷霆之体：+20%
- 五行灵体：+25%
- 凤凰涅槃：+30%

#### 传承加成
- 剑意觉醒：+20%
- 仙人传承：+40%
- 上古传承：+30%

#### 负面状态
- 道心破裂：-20%
- 诅咒法器：-10%

---

### 2. 战斗难度评估

系统根据战力比值自动评估：

| 战力比值 | 难度等级 | 胜率范围 | 描述 |
|---------|---------|---------|------|
| ≥2.0 | 轻而易举 | 90%+ | 碾压级别 |
| 1.3-2.0 | 胜券在握 | 75-90% | 明显优势 |
| 0.8-1.3 | 势均力敌 | 50-75% | 正常战斗 |
| 0.5-0.8 | 凶险万分 | 25-50% | 处于劣势 |
| 0.3-0.5 | 九死一生 | 10-25% | 极度危险 |
| <0.3 | 必死无疑 | <10% | 几乎必败 |

---

### 3. 战斗奖励与惩罚

#### 胜利奖励（根据难度）

| 难度 | 修为 | 灵石 | 寿命消耗 |
|-----|------|------|---------|
| 轻而易举 | +20 | +30 | -1 |
| 胜券在握 | +50 | +80 | -1 |
| 势均力敌 | +100 | +150 | -2 |
| 凶险万分 | +150 | +220 | -4 |
| 九死一生 | +220 | +300 | -6 |
| 必死无疑 | +300 | +500 | -8 |

**设计理念**：越困难的战斗，胜利后收获越大

#### 失败惩罚（根据难度）

| 难度 | 修为 | 灵石 | 寿命损失 |
|-----|------|------|---------|
| 轻而易举 | -30 | -50 | -3 |
| 胜券在握 | -20 | -40 | -4 |
| 势均力敌 | -10 | -30 | -5 |
| 凶险万分 | -5 | -20 | -8 |
| 九死一生 | 0 | -10 | -12 |
| 必死无疑 | 0 | 0 | -20 |

**设计理念**：
- 输给弱者：羞辱性惩罚（修为倒退）
- 输给强者：能活下来就不错了（重伤但修为不退）

## 新增战斗事件

### 1. 妖兽袭击
- **敌人**：凶猛妖兽（战力150）
- **选项**：评估实力 / 迎战 / 逃跑

### 2. 散修挑战
- **敌人**：同境界散修（战力90%）
- **选项**：接受决斗 / 和解（50灵石）/ 拒绝（损声望）

### 3. 宝藏守护者
- **敌人**：守护灵兽（筑基期，战力150%）
- **胜利**：获得300灵石宝藏
- **选项**：挑战 / 偷窃（30%成功）/ 放弃

### 4. 魔修伏击
- **敌人**：魔修小队（筑基期，战力200%）
- **选项**：以一敌多 / 遁符逃脱（100灵石）/ 交出资源（150灵石）

### 5. 宗门试炼
- **选项**：
  - 挑战弱者（战力70%）- 稳妥但收获少
  - 挑战同境界（战力100%）- 正常收益
  - 挑战强者（筑基期，战力120%）- 以弱胜强，名扬宗门

### 6. 带伤遇敌
- **特点**：寿命低时战力大幅下降
- **选项**：带伤迎战 / 虚张声势（40%成功）/ 求饶

### 7. 兽潮来袭
- **敌人**：妖兽兽潮（筑基期，战力300%）
- **胜利**：声望+60，灵石+500，获得"兽潮英雄"称号

### 8. 法宝对决
- **敌人**：持宝修士（筑基期，战力180%）
- **特殊**：如果自己有法器，敌人战力降至120%

### 核心方法
```typescript
// 计算玩家战力
calculatePlayerPower(): number

```

## 游戏策略影响

### 1. 状态管理更重要
- 寿命低时避免战斗
- 保持善缘提升战力
- 及时治疗负面状态

### 2. 装备价值提升
- 法器不再是摆设，直接影响战力
- 体质、传承的战略价值凸显

### 3. 风险与收益平衡
- 挑战强敌：高风险高回报
- 碾压弱者：安全但收益少
- 势均力敌：最佳性价比

### 4. 战术选择
- 评估实力后再决定
- 可以花钱避免战斗
- 逃跑也是一种策略

## 测试建议

1. **低寿命测试**：寿命<20%时挑战敌人，验证战力下降
2. **装备测试**：获得法器前后对比战力
3. **难度测试**：挑战不同难度的敌人，验证胜率
4. **边界测试**：战力悬殊时的战斗结果
5. **连续战斗**：多次战斗后寿命消耗的累积效果
