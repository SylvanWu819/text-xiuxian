## 问题排查

### 如果按钮仍然无响应

#### 检查 1：元素是否存在
在控制台中运行：
```javascript
console.log('btn-new-game:', document.getElementById('btn-new-game'));
console.log('btn-continue-game:', document.getElementById('btn-continue-game'));
console.log('btn-how-to-play:', document.getElementById('btn-how-to-play'));
```

应该看到三个按钮元素，而不是 `null`。

#### 检查 2：事件监听器是否绑定
在控制台中运行：
```javascript
const btn = document.getElementById('btn-new-game');
console.log('事件监听器:', getEventListeners(btn));
```

应该看到 `click` 事件监听器。

#### 检查 3：CSS 是否阻止点击
检查按钮是否被其他元素覆盖：
```javascript
const btn = document.getElementById('btn-new-game');
console.log('按钮位置:', btn.getBoundingClientRect());
console.log('pointer-events:', window.getComputedStyle(btn).pointerEvents);
```

`pointer-events` 应该是 `auto`，不应该是 `none`。

#### 检查 4：JavaScript 错误
查看控制台是否有任何 JavaScript 错误。如果有错误，可能会阻止事件监听器的绑定。
