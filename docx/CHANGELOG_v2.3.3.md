# 修仙模拟器 v2.3.3 更新日志

## 📅 发布日期
2026-04-30

## 🎯 重大改进

本次更新修复了**9个关键逻辑问题**，大幅提升游戏稳定性和平衡性。

---

## 🐛 核心逻辑修复

### 1. 修为突破系统
**问题**：突破失败时修为降至80%，可能仍≥突破阈值，导致无限循环
**修复**：
- 失败时修为降至50%，彻底避免循环
- 因果影响从±50%改为±30%，更合理
- 最低成功率从10%提升到15%

### 2. 战力计算系统
**问题**：多个标记加成相乘，导致战力指数增长（最高可达2.275倍）
**修复**：
- 改为加法计算，避免指数增长
- 总加成限制在-50%到+200%之间
- 战力增长更线性可控

### 3. 战斗寿命消耗
**问题**：胜利也消耗大量寿命（1-8年），频繁战斗快速耗尽寿命
**修复**：
- 轻松胜利（VeryEasy/Easy）不消耗寿命
- 正常战斗消耗1年
- 困难战斗消耗2-5年（原6-8年）
- 鼓励玩家主动战斗

### 4. 声望系统同步
**问题**：通过事件获得声望时，正道/魔道相互抵消逻辑不生效
**修复**：
- 正道行为调用`performRighteousAction()`，自动降低魔道声望
- 魔道行为调用`performDemonicAction()`，自动降低正道声望
- 声望变化更符合逻辑

### 5. 寿命耗尽检查
**问题**：寿命耗尽时游戏继续运行，未触发结局
**修复**：
- 时间推进后检查寿命是否耗尽
- 寿命为0时自动触发"寿命耗尽"结局
- 寿命变化时显示警告提示

### 6. 存档版本兼容
**问题**：2.3.1存档无法在2.3.2加载（补丁版本不兼容）
**修复**：
- 允许同一主次版本的补丁版本差异
- 2.3.x的所有存档互相兼容
- 向后兼容性更好

### 7. 选项要求检查
**问题**：未检查`requiredFlags`，某些需要剧情标记的选项被错误允许
**修复**：
- 添加剧情标记检查
- 返回详细错误信息
- 防止玩家触发不应该出现的选项

### 8. 修为边界检查
**问题**：修为可能变为负数
**修复**：
- 修为变化后检查是否<0
- 自动修正为0
- 防止数值异常

### 9. 寿命警告提示
**问题**：寿命即将耗尽时无提示
**修复**：
- 寿命减少后检查是否耗尽
- 显示"⚠️ 警告：你的寿命即将耗尽！"
- 给玩家反应时间
