# 实现计划：修仙模拟器

## 概述

本实现计划将修仙模拟器从设计转化为可运行的 VSCode 插件。采用前后端分离架构，Extension 后端处理游戏逻辑，Webview 前端负责界面渲染。实现将按照从基础架构到核心系统再到前端界面的顺序进行，确保每个步骤都能增量验证功能。

## 技术栈

- **后端**: TypeScript + VSCode Extension API
- **前端**: HTML + CSS + Vanilla JavaScript
- **存储**: VSCode globalState API
- **配置**: JSON 文件
- **测试**: Jest 或 Mocha（可选）
